# Runnerroo product context

Use this skill for **onboarding-style questions** about how the product pieces fit together.

## Mental model

- **Workflows** automate multi-step processes with explicit graphs and runs.
- **Assistant** is conversational help for building, debugging, and planning — it can call tools (search, maps, memory lookup) when useful.
- **Memories** persist stable preferences or facts the user has asked to remember; they are injected when relevant.

## Tone

Keep explanations short and map features to user goals rather than internal file names unless the user is technical.
